# 四川大学课程余量查询工具

想知道一门课还有没有名额、某位老师开了哪些课，或者某个学期的课容量是多少？这个工具会登录四川大学 URP 教务系统，按课程名或教师名查询课程，并在接口可用时显示课容量、课余量和已选人数。

它不会替你选课，只负责查询。

## 适合做什么

- 选课前后快速查某门课还有多少余量
- 按课程关键词查所有匹配课程
- 按教师姓名查授课列表
- 查看课程号、课序号、教师、学分、学时、校区、教室、上课周次等详情
- 在不同学期之间快速切换查询目标

## 最快开始

推荐使用 `uv` 启动。第一次运行前先同步依赖：

```powershell
uv sync
```

复制配置模板：

```powershell
Copy-Item .\config.json.example .\config.json
```

编辑 `config.json`：

```json
{
  "username": "你的学号",
  "password": "你的密码",
  "zxjxjhh": "2025-2026-2-1"
}
```

启动交互查询：

```powershell
uv run query
```

进入后直接输入课程关键词，例如：

```text
计算机
```

使用 `/term` 命令设置要查询的学期，例如查询本学期：

```text
/term now
```

如果暂时不使用 `uv`，也可以用 pip 安装依赖后启动：

```powershell
python -m pip install -r requirements.txt
python .\run.py
```

## 查询效果

当课表接口返回容量数据时，列表会显示类似内容：

```text
序号  课程号        课序号  课程名称              教师      课容量  课余量  已选
1     304036030    01      计算机组成原理        李征      120     12      108
2     304036030    02      计算机组成原理        罗川      120     0       120
```

字段含义：

- `课容量`：这门课总共可容纳的人数
- `课余量`：当前剩余名额
- `已选`：选课页面“点击查看”接口返回的实时已选人数

输入 `d1` 可以查看第 1 条结果的详细信息（学分、学时、校区、教室、上课周次、课程属性等）。

输入 `d2` 可以查看第2条结果的详细信息。

## 常用命令

| 输入 | 作用 |
| --- | --- |
| `计算机` | 按课程名称关键词查询 |
| `-t 张三` | 按授课教师姓名查询 |
| `d1` | 查看上一轮结果中的第 1 门课详情 |
| `/term` | 查看当前学期和学期切换帮助 |
| `/term now` | 自动切到当前日期对应学期 |
| `/term 25-2` | 切到 `2025-2026` 学年第 2 学期 |
| `/term next` | 切到下一学期 |
| `/term prev` | 切到上一学期 |
| `q` | 退出程序 |

也可以跳过交互模式，直接查课程名：

```powershell
uv run query -k "计算机"
```

查看命令行帮助：

```powershell
uv run query -h
```

## 学期切换

URP 查询接口使用的学期号格式是：

```text
YYYY-YYYY-X-1
```

例子：

```text
2025-2026-1-1   2025-2026 学年第 1 学期
2025-2026-2-1   2025-2026 学年第 2 学期
```

为了不用每次输入完整学期号，交互模式支持短命令：

```text
/term 25-2
```

等价于：

```text
2025-2026-2-1
```

也可以使用完整写法：

```text
/term 2025-2026-2-1
```

切换成功后，程序会把新的 `zxjxjhh` 写入本地 `config.json`，下次启动继续使用。

## 配置文件

`config.json` 是你的本地配置，包含账号密码，不会提交到仓库。

字段说明：

| 字段 | 说明 |
| --- | --- |
| `username` | URP 登录账号，通常是学号 |
| `password` | URP 登录密码 |
| `zxjxjhh` | 查询学期号，可选；也可以在程序里用 `/term` 设置 |

仓库只提供安全模板：

```text
config.json.example
```

## 查询接口

当前优先使用自由选课接口 `/student/courseSelect/freeCourse/courseList` 查询课程列表和课余量。
选课页面中的已选人数来自另一个接口 `/student/courseSelect/selectCourse/viewXkCount/{学期号}/{课程号}/{课序号}`，程序会在选课接口可用时自动补充这个字段。

代码里也保留了教学资源接口作为注释，教学资源接口通常只返回课程基础信息，不返回课容量和课余量。

## 注意事项

- 请不要把真实 `config.json` 上传到公共仓库。
- 请避免高频请求教务系统。
- 验证码由 `ddddocr` 自动识别，偶尔失败时程序会自动重试。
- 如果查不到课程，优先确认当前学期是否正确：进入程序后输入 `/term` 查看。
- 教务系统接口可能变化；如果返回字段改变，需要同步更新解析逻辑。
